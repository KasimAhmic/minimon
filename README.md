## Installing

1. Install Node.js
2. Install Yarn
3. Clone the repo
   - `git clone https://github.com/KasimAhmic/minimon.git`
4. Install dependencies
   - `yarn run init`
5. Start Minimon
   - `yarn start`
6. Navigate to `http://localhost:3000` on the host machine or `http://YOUR_HOST_MACHINE'S_LOCAL_IP:3000` on another device
